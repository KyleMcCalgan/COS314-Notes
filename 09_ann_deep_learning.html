ANN properties, neuron models, perceptrons, multilayer networks, and the deep-learning concepts named in the final scope.

Computational Neuron

An artificial neuron computes a weighted sum of inputs, adds a bias, then applies an activation function.

n = w1*p1 + w2*p2 + ... + wn*pn + b
a = f(n)
PartRole
Input pThe feature vector supplied to the neuron.
Weights wLearned values that control how strongly each input contributes.
Bias bLearned offset that shifts the activation boundary.
Activation fTransforms the weighted sum into the neuron's output.

MCP Neuron and Linear Separability

The McCulloch-Pitts neuron uses binary inputs, weights, a threshold, and a step activation. It can implement simple linearly separable logic such as AND and OR.

f(n) = 1 if n >= theta, else 0
OR with w1 = 2, w2 = 2, theta = 2 Inputs 01, 10, and 11 reach the threshold. Input 00 does not.
AND with w1 = 1, w2 = 1, theta = 2 Only input 11 reaches the threshold.
A single MCP neuron cannot solve XOR because XOR is not linearly separable. A multilayer network is needed for non-linear decision boundaries.

Perceptron

A perceptron is a supervised feedforward classifier. It learns weights and bias by repeatedly checking training instances and updating when the output is wrong.

PERCEPTRON LEARNING initialise weights and bias repeat until convergence: for each training instance p with target t: n = w dot p + b output = f(n) if output != t: w_i = w_i + alpha * (t - output) * p_i b = b + alpha * (t - output)
PropertyMeaning
FeedforwardSignals move from input to output without feedback loops.
Converges when possibleIf the data is linearly separable, learning can settle on a separating boundary.
LimitationA single layer cannot solve non-linearly separable problems such as XOR.

MLP Architecture

A multilayer perceptron has an input layer, one or more hidden layers, and an output layer. Hidden layers let the network learn non-linear mappings.

Input layer Hidden layer Output layer p1 p2 p3 -> h1 h2 ... -> output nodes
Depth Number of computational layers, usually hidden layers plus output layer. More depth can model more complex transformations.
Width Number of neurons in a layer. Too little width can underfit; too much can overfit.
Model scaling Modern deep models usually scale both depth and width. Making a model deeper without enough width can create information bottlenecks.

Deep Learning

A deep neural network has many hidden layers. The distinction from a standard multilayer network is mainly scale: deep networks may have dozens or hundreds of layers.

OptimiserUses gradients to update weights and reduce loss.
OverfittingHigh training accuracy but poor test accuracy.
Transfer learningReuse weights/features from a model pretrained on another dataset.

Overfitting Controls

Regularisation Adds a penalty for large weights so the model is less likely to memorise noise.
Early stopping Stop training when validation performance starts getting worse.
Dropout Randomly deactivate neurons during training so the model learns robust representations.
Reduce model size Fewer layers or neurons can reduce excessive capacity.
