K-means Clustering

Unsupervised clustering, centroids, Euclidean distance, assignment/update iterations, and convergence.

Unsupervised Learning Context

Clustering is unsupervised learning: the dataset has no class labels, and the algorithm groups similar instances together.

Classification Supervised. The model predicts known class labels.
Clustering Unsupervised. The algorithm discovers groups from similarity.

K-means Core Idea

K-means clusters a numerical dataset into K clusters. Each cluster is represented by a centroid, which is the mean of the points assigned to that cluster.

TermMeaning
KThe number of clusters to create.
CentroidThe average point of all instances currently assigned to a cluster.
Assignment phaseAssign each instance to the nearest centroid.
Update phaseRecalculate each centroid from its assigned instances.
ConvergenceNo instances move between clusters after an iteration.
The standard algorithm requires numerical data because it uses distances and averages.

Algorithm

K-MEANS choose K randomly select K initial centroids while clusters have not converged: for each data instance: calculate distance to every centroid assign instance to the closest centroid for each cluster: update centroid to the average of assigned instances return final clusters and centroids
Euclidean distance = sqrt((x1 - y1)^2 + (x2 - y2)^2 + ... + (xm - ym)^2)
Rounding rule from exam notice Round to 2 decimal places at every step using standard 0.5 rounding. Do not carry unrounded figures into later steps.

Advantages and Limitations

Advantages Simple, fast, intuitive, and often converges quickly when clusters are well separated.
Limitations Sensitive to initial centroids, outliers, and the chosen value of K. Assumes roughly spherical clusters and numeric attributes.
