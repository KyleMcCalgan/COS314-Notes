Backpropagation

Forward pass, loss calculation, backward pass, optimiser updates, and the formal training loop.

Training Cycle

Backpropagation is the algorithm that calculates how much each weight contributed to the loss. The optimiser then uses those gradients to update the weights and biases.

Step 1
Forward pass
Input data moves through the network to produce an output.
v
Step 2
Loss calculation
Compare prediction with target and compute the error.
v
Step 3
Backpropagation
Propagate error backward and compute gradients using the chain rule.
v
Step 4
Optimisation
Update weights and biases to reduce future loss.
Backpropagation finds the gradients. The optimiser applies the update. Do not treat those as the same thing.

Formal Algorithm

BACKPROPAGATION LEARNING initialise weights and biases to small random values while stopping condition is not met: perform feedforward learning compute output error and hidden-layer error update output-layer weights and bias update hidden-layer weights and bias return trained network
Stopping conditionMeaning
Error no longer changesThe network has converged or reached a plateau.
Error increasesTraining may be overfitting or diverging.
Maximum epochsA fixed training budget has been reached.

Feedforward Detail

For a network with one hidden layer, compute hidden-node weighted sums first, activate them, then compute output-node weighted sums and activations.

n1_j = v0_j + sum(v_lj * p_l)
h_j = f(n1_j)
n2_m = w0_m + sum(w_im * h_i)
output_m = f(n2_m)
Key point The output layer receives the activated hidden outputs h_i, not the raw hidden weighted sums n1_i.

Sigmoid Derivatives

ActivationFunctionDerivative
Binary sigmoidf(n) = 1 / (1 + e^-n)f'(n) = f(n)(1 - f(n))
Bipolar sigmoidf(n) = 2 / (1 + e^-n) - 1f'(n) = 0.5(1 + f(n))(1 - f(n))
These forms are convenient because the derivative can be computed from the activation output itself.

Output And Hidden Error

The exact notation varies by slide set, but the structure is stable: calculate output-layer deltas first, then use those deltas to calculate hidden-layer deltas.

output delta = (target - output) * f'(output net input)
hidden delta = f'(hidden net input) * sum(output delta * outgoing weight)
Output layer Directly compare target and output because the target values are known.
Hidden layer No direct target exists, so error is assigned through the weighted contribution to output errors.

Weight Update Pattern

new weight = old weight + alpha * delta * input_to_that_weight
new bias = old bias + alpha * delta
SymbolMeaning
alphaLearning rate. Controls the step size of updates.
deltaError signal for the destination neuron.
input_to_that_weightThe activation feeding into the weight being updated.
Learning rate effect Too large can overshoot the minimum. Too small can make training very slow or stuck.

Optimisers

OptimiserIdea
Gradient descentMove weights opposite the gradient of the loss.
Stochastic gradient descentUpdate using one training example at a time.
Mini-batch gradient descentUpdate using a small batch of examples.
MomentumUse memory of previous gradient direction.
AdamCombines momentum-style behaviour with adaptive per-weight learning rates.
